## Architecture Highlights

Built with **Clean Architecture** and industry best practices:

- **State Management**: BLoC pattern with `flutter_bloc` for predictable state updates
- **Dependency Injection**: `get_it` for decoupled, testable components
- **Functional Programming**: `fpdart` for type-safe error handling with `Either<Failure, T>`
- **Feature-First Structure**: Modular codebase organized by business domain
