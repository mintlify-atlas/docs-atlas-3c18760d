## Overview

The Barcode Scanning feature uses the device camera to detect and scan product barcodes in real-time. It includes smart cooldown logic to prevent duplicate scans and provides haptic feedback for better user experience.

## Implementation

The scanner is implemented on the home page using the `mobile_scanner` package:

### Scanner Controller Setup

```dart lib/features/billing/presentation/pages/home_page.dart
final MobileScannerController _scannerController = MobileScannerController(
  detectionSpeed: DetectionSpeed.normal,
  returnImage: false,
);

bool _isCameraOn = true;
bool _isFlashOn = false;

// Cooldown mapping to prevent rapid firing of the same barcode
final Map<String, DateTime> _lastScanTimes = {};
```

## Barcode Detection

### Detection Handler with Cooldown

The `_onDetect` method processes barcodes with intelligent cooldown logic:

```dart lib/features/billing/presentation/pages/home_page.dart
void _onDetect(BarcodeCapture capture) async {
  final List<Barcode> barcodes = capture.barcodes;
  final now = DateTime.now();

  for (final barcode in barcodes) {
    if (barcode.rawValue != null) {
      final rawValue = barcode.rawValue!;

      // Cooldown logic: 2 seconds per identical barcode
      if (_lastScanTimes.containsKey(rawValue)) {
        final lastScan = _lastScanTimes[rawValue]!;
        if (now.difference(lastScan).inSeconds < 2) {
          continue; // Skip this scan
        }
      }

      _lastScanTimes[rawValue] = now;

      // Vibrate
      final hasVibrator = await Vibration.hasVibrator();
      if (hasVibrator == true) {
        Vibration.vibrate();
      }

      if (mounted) {
        context.read<BillingBloc>().add(ScanBarcodeEvent(rawValue));
      }
      break; // Process one barcode at a time per frame
    }
  }
}
```

<Note>
The 2-second cooldown prevents accidental duplicate scans while still allowing intentional re-scans of the same product.
</Note>

### Why Cooldown is Important

<AccordionGroup>
  <Accordion title="Prevents Duplicate Entries">
    Without cooldown, a single barcode can be detected multiple times per second, causing unwanted duplicate cart entries.
  </Accordion>
  
  <Accordion title="Improves User Experience">
    Users get clear feedback that a product was scanned without overwhelming them with rapid-fire additions.
  </Accordion>
  
  <Accordion title="Allows Intentional Re-scans">
    The 2-second window is short enough that users can deliberately scan the same product again if needed.
  </Accordion>
</AccordionGroup>

### Corner Indicators

```dart lib/features/billing/presentation/pages/home_page.dart
Widget _buildCorner(Alignment alignment) {
  return Align(
    alignment: alignment,
    child: Container(
      width: 32,
      height: 32,
      decoration: BoxDecoration(
        border: Border(
          top: (alignment == Alignment.topLeft ||
                  alignment == Alignment.topRight)
              ? const BorderSide(color: Colors.greenAccent, width: 4)
              : BorderSide.none,
          bottom: (alignment == Alignment.bottomLeft ||
                  alignment == Alignment.bottomRight)
              ? const BorderSide(color: Colors.greenAccent, width: 4)
              : BorderSide.none,
          left: (alignment == Alignment.topLeft ||
                  alignment == Alignment.bottomLeft)
              ? const BorderSide(color: Colors.greenAccent, width: 4)
              : BorderSide.none,
          right: (alignment == Alignment.topRight ||
                  alignment == Alignment.bottomRight)
              ? const BorderSide(color: Colors.greenAccent, width: 4)
              : BorderSide.none,
        ),
      ),
    ),
  );
}
```

## Camera Controls

### Overlay Buttons

```dart lib/features/billing/presentation/pages/home_page.dart
Positioned(
  top: MediaQuery.of(context).padding.top + 16,
  right: 16,
  child: Column(
    children: [
      _buildOverlayButton(
        icon: Icons.settings,
        onPressed: () async {
          _scannerController.stop();
          await context.push('/settings');
          if (_isCameraOn && mounted) _scannerController.start();
        },
      ),
      const SizedBox(height: 16),
      if (_isCameraOn)
        _buildOverlayButton(
          icon: _isFlashOn ? Icons.flashlight_off : Icons.flashlight_on,
          onPressed: () {
            setState(() => _isFlashOn = !_isFlashOn);
            _scannerController.toggleTorch();
          },
        ),
      if (_isCameraOn) const SizedBox(height: 16),
      _buildOverlayButton(
        icon: _isCameraOn ? Icons.videocam : Icons.videocam_off,
        onPressed: () {
          setState(() {
            _isCameraOn = !_isCameraOn;
          });
          if (_isCameraOn) {
            _scannerController.start();
          } else {
            _scannerController.stop();
          }
        },
      ),
    ],
  ),
)
```

## Scanner Lifecycle Management

### Stopping on Navigation

```dart

  _scannerController.stop();
  await context.push('/checkout');
  if (_isCameraOn && mounted) _scannerController.start();

```

### Cleanup on Disposal

```dart
@override
void dispose() {
  _scannerController.dispose();
  super.dispose();
}
```

<Warning>
Always stop the scanner when navigating away and dispose of the controller to prevent memory leaks and battery drain.
</Warning>

## Best Practices

<AccordionGroup>
  <Accordion title="Optimize Performance">
    - Set `returnImage: false` if you don't need barcode images
    - Use `DetectionSpeed.normal` for balanced performance
    - Stop the scanner when not in use
  </Accordion>
  
  <Accordion title="Improve Accuracy">
    - Provide clear visual feedback (bounding box, corners)
    - Add sufficient lighting or enable flashlight
    - Ensure barcodes are not damaged or obscured
  </Accordion>
  
  <Accordion title="Handle Edge Cases">
    - Check if device has vibration before using it
    - Validate barcode format before processing
    - Handle "product not found" gracefully
  </Accordion>
</AccordionGroup>
